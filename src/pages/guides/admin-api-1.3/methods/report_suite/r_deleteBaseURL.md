# DeleteBaseURL

Deletes the Base URL for each of the specified report suites.

## ReportSuite.DeleteBaseURL parameters

|Name|Type|Description|
|----|----|-----------|
|**rsid\_list** |`array(xsd:string)` |A list of report suite IDs.|

## ReportSuite.DeleteBaseURL response

|Type|Description|
|----|-----------|
|`xsd:int` |Returns `1` if the operation is successful. Otherwise, returns `0`.|
