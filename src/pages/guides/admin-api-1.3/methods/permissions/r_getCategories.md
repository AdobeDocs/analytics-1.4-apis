# GetCategories

Returns the permissions category structure and indicates if each permission is selected.

## Permissions.GetCategories parameters

|Name|Type|Description|
|----|----|-----------|
|**groupid** |`xsd:int` |The group ID.|
|**categoryid** |`xsd:int` |The category ID.|

## Permissions.GetCategories response

|Type|Description|
|----|-----------|
|[tns:perm\_categories](../../data_types/r_perm_categories.md#) |A list of a company's permission categories.|
