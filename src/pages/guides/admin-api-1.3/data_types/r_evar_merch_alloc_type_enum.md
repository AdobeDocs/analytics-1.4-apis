# evar\_merch\_alloc\_type\_enum

An enumerated list of supported values for merchandising allocation of success events.

The enumerated type includes the following elements of type `xsd:string`.

|Name|Description|
|----|-----------|
|**0** | Allocate to last. |
|**1** | Allocate to first. |
