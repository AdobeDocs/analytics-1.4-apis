# evar\_exp\_type\_enum

An enumerated list of supported eVar expiration values.

The enumerated type includes the following elements of type `xsd:string`.

|Name|Description|
|----|-----------|
|**0** | Visit |
|**1** | Page View |
|**2** | Never |
|**3** | Minute |
|**4** | Hour |
|**5** | Day |
|**6** | Week |
|**7** | Month |
|**8** | Quarter |
|**9** | Year |
|**10** | Purchase |
|**11** | Product View |
|**12** | Cart Open |
|**13** | Cart Checkout |
|**14** | Cart Add |
|**15** | Cart Remove |
|**16** | Cart View |
