# status

Data structure that contains information about the status of a data source.

|Element|Type|Description|
|-------|----|-----------|
|**statusCode** |`int` | The data source status code. |
|**message** |`string` | A brief message associated with the current data source status. See [Status Codes](../c_status_codes.md#). |

**Parent topic:** [Data Types](../data_types/c_data_types.md)
