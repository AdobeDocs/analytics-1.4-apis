# list_variable_expiration_type

An enumerated list of expiration types.

One of the following values of type `xsd:string`:

- custom
- visit
- page_view
- never
- minute
- hour
- day
- week
- month
- quarter
- year
- purchase
- product_view
- cart_open
- cart_checkout
- cart_add
- cart_remove
- cart_view
