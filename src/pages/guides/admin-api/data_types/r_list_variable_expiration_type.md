# list\_variable\_expiration\_type

An enumerated list of expiration types.

One of the following values of type `xsd:string`:

-   custom
-   visit
-   page\_view
-   never
-   minute
-   hour
-   day
-   week
-   month
-   quarter
-   year
-   purchase
-   product\_view
-   cart\_open
-   cart\_checkout
-   cart\_add
-   cart\_remove
-   cart\_view
