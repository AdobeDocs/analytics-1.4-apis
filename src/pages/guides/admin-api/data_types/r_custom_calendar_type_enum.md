# custom\_calendar\_type\_enum

An enumerated list of custom calendar types.

The enumerated type includes the following elements of type `xsd:string`.

-   gregorian
-   4-5-4-retail
-   4-5-4-custom
-   4-4-5-custom
-   modified-gregorian
